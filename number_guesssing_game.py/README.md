
# 🎯 Number Guessing Game (CustomTkinter GUI)

A visually engaging number guessing game built using **CustomTkinter**, featuring a slot-machine-style digit display, limited lives, and interactive feedback.

---

## 🚀 Features

* 🎮 Guess a number between **1 and 100**
* ❤️ Limited attempts (**7 lives**)
* 🔢 Dynamic two-digit display
* 📉 Feedback system:

  * Too High
  * Too Low
  * Correct Guess
* 🔄 Smooth reset and replay flow
* 🎨 Clean, themed UI

---

## 🧠 Game Logic

* A random number between **1 and 100** is generated at the start
* Each guess updates display and reduces lives
* Game ends when:

  * Correct guess
  * Lives reach zero

---

## ▶️ Usage

```bash
python number_guessing_game.py
```

---

## 🛠️ Tech Stack

* Python
* CustomTkinter
* Random module
