
# 🧠 Python Simple Projects Collection

A collection of beginner-to-intermediate Python projects focused on building core programming skills through hands-on implementation.

This repository emphasizes:

* Problem-solving
* Event-driven programming
* Data handling
* Incremental learning through small systems

---


## 🚀 Projects Overview

### 🎯 Number Guessing Game

A GUI-based game where the player guesses a number within limited attempts.

**Concepts:**

* State management
* Input validation
* UI updates
* Game lifecycle handling

---

### 🪨✋✌️ Rock Paper Scissors

A timed GUI game where the player competes against the computer.

**Concepts:**

* Event-driven programming (`after`)
* Timers and delays
* Game logic mapping
* UI state control

---

### 📝 To-Do List (CLI)

A command-line task manager with persistent storage.

**Concepts:**

* CRUD operations
* JSON-based persistence
* File handling
* Data modeling

---

### 🔐 Caesar Cipher

A simple encryption/decryption tool using character shifting.

**Concepts:**

* ASCII manipulation (`ord`, `chr`)
* Modular arithmetic
* String processing

---

## 🛠️ Tech Stack

* **Python**
* **CustomTkinter** (for GUI projects)
* **JSON** (data persistence)
* **Standard Library** (`random`, `os`, `pathlib`)

## ▶️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/your-username/python-simple-projects.git
cd python-simple-projects
```

---

### 2. Install dependencies

```bash
pip install customtkinter
```

---

### 3. Run any project

Example:

```bash
python number_guessing_game.py
```

or

```bash
python to_do_list/to_do_list.py
```
