# 📝 CLI Task Manager

A simple command-line task management system with persistent storage using JSON.

---

## 🚀 Features

* ➕ Add tasks
* 📋 List all tasks
* 🔄 Update task status
* ❌ Delete tasks
* 💾 Save tasks to file

---

## 🧠 How It Works

* Tasks are stored as:

  * ID
  * Description
  * Status

* Data is saved in `tasks.json` and loaded on startup

---

## ▶️ Usage

```bash
python to_do_list/to_do_list.py
```

---

## 🛠️ Tech Stack

* Python
* JSON
* Pathlib / OS
