# 🪨✋✌️ Rock Paper Scissors (CustomTkinter GUI)

A modern GUI-based Rock-Paper-Scissors game with a timer-based decision system and real-time score tracking.

---

## 🚀 Features

* 🎮 Interactive gameplay
* ⏳ Countdown timer per round
* 🤖 Auto-scoring if time runs out
* 📊 Live score tracking
* 🔄 Restart functionality
* 🎨 Styled UI with custom colors

---

## 🧠 Game Logic

* Player must choose within time limit
* If no input → computer gains a point
* Standard rules apply:

  * Rock beats Scissor
  * Scissor beats Paper
  * Paper beats Rock

---

## ▶️ Usage

```bash
python rock_paper_scissors.py
```
