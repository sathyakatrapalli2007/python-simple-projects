# 🔐 Caesar Cipher (Python)

A simple implementation of the Caesar Cipher for encrypting and decrypting text using character shifting.

---

## 🚀 Features

* 🔒 Encrypt text
* 🔓 Decrypt text
* 🔤 Supports uppercase and lowercase
* 🔁 Wrap-around using modulo arithmetic

---

## 🧠 How It Works

* Each character is shifted by a fixed key
* Uses ASCII values (`ord`, `chr`)
* Applies modulo 26 to stay within alphabet

---

## ▶️ Usage

```bash
python caeser_cipher.py
```
